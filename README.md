# MCP Abilities - Plugin Check

MCP bridge for the official WordPress.org Plugin Check plugin.

This plugin exposes Plugin Check results through an authenticated MCP ability so release workflows can run the same checks from automation.

## Abilities

- `plugin-check/run`

## Requirements

- WordPress 6.9+
- PHP 8.0+
- Abilities API
- MCP Adapter
- Plugin Check

## Notes

The official Plugin Check plugin must be installed and active on the target WordPress site.
